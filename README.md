## Welcome to the simplest user-friendly player ever made!

### Applicable preferences:
- Label (String)
- AspectRatio (double)
- AutoPlay (bool)
- LoopMode (bool)
- ForceAspectRatio (bool)
- ColorAccent (Color)

</br><div>

```dart
//Example:
SimplePlayer.build(
    simplePlayerSettings: SimplePlayerSettings.network(
        path: url,
        label: 'Bee',
        aspectRatio: 16 / 9,
        autoPlay: false,
        loopMode: false,
        forceAspectRatio: false,
        colorAccent: Colors.red,
    ),
),

SimplePlayer.build(
    simplePlayerSettings: SimplePlayerSettings.assets(
        path: url,
        label: 'Bee',
        aspectRatio: 16 / 9,
        autoPlay: false,
        loopMode: false,
        forceAspectRatio: false,
        colorAccent: Colors.red,
    ),
),
```
